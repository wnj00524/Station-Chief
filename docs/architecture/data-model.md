# Data Model

The slice should be data-driven.

## Required record types
- `Nominal`
- `Location`
- `EmailMessage`
- `InterceptRecord`
- `StaffMember`
- `Task`
- `WorldSignal`
- `CaseDefinition`
- `OutcomeDefinition`

## Schema guidance
Keep narrative content out of system code. Store authored text and branching configuration in content files.
