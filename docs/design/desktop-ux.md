# Desktop UX Targets

## Required applications for the slice
- Inbox
- Nominals / Database
- Intercepts
- Map
- Staff tasking panel

## UX rules
- Strong dark-mode readability
- Windowed workflow that encourages comparison across sources
- Visible timestamps and recurring names/locations across apps
- No objective markers or non-diegetic prompts
- Action buttons only where they logically belong inside the interface

## Top bar
- Local / global clock
- Connection strength indicator
- Political Capital counter

## Dock
Left-side launcher with app entry points.
