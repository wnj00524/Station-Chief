# Game Pillars

## 1. Pure simulation
The game should not solve the mystery for the player. Evidence must be internally consistent, but the player must notice contradictions themselves.

## 2. Diegetic desktop
The desktop is not a wrapper around the game. It is the game board.

## 3. Pressure from time
Reading, verifying, and hesitating all consume time. Investigation choices must compete with unfolding events.

## 4. Pragmatic factions
No faction should behave like a purely moral caricature. Each actor pursues leverage, survival, legitimacy, or return on investment.
