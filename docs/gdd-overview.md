# GDD Overview

## Project
**STATION_CHIEF** is a hardcore intelligence simulation presented entirely through a virtual desktop interface. The player is a foreign intelligence officer working from a regional station office.

## North-star pillars
- **Pure Simulation** — no quest markers; the player notices discrepancies manually.
- **Diegetic Interface** — if it is not on the desktop, it does not exist.
- **Real-Time Pressure** — the clock continues while the player investigates.
- **Stability Agnosticism** — rival agencies are pragmatic actors with competing interests, not simple villains.

## Core player fantasy
The player receives incomplete or conflicting information, cross-references records across desktop apps, forms a hypothesis, acts under pressure, and lives with the consequences.

## Core loop
1. Collection
2. Processing
3. Correlation
4. Dissemination / Action

## Vertical slice implication
The first playable build should prove that this loop is fun and legible before building broad simulation depth or heavy UI polish.
